# 📌 Python Calculator & Math Challenge

A simple command-line project built in Python as a collaborative learning project.
This program includes a calculator and a multiplication challenge game to practice math skills.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Input validation

## 🎮 Multiplication Challenge

- Random multiplication questions
- Two difficulty levels:
    - Easy (numbers 1–5)
    - Hard (numbers 6–9)
- Answer validation

## 🛠 Technologies

- Python 3
- Git
- GitHub

## 📂 Project Structure

```
python-calculator/ 
│ 
├── main.py            # Main menu of the application 
├── calculator.py      # Calculator menu and logic 
├── operations.py      # Mathematical operations 
├── math_game.py       # Multiplication challenge game 
└── README.md
```

## ▶ How to Run
```
python main.py
```

Then choose one option from the main menu:

1 - Calculator
2 - Multiplication Challenge
0 - Exit
